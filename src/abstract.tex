\section*{Resumo}

Este projeto demonstra uma organizacao modular em \LaTeX{} usando subarquivos
standalone, bibliografia no padrao ABNT com \texttt{biber} e um apendice para
registro de strings de busca. A estrutura foi separada para manter o arquivo
principal enxuto e concentrar configuracoes especificas perto do conteudo que as
utiliza.
